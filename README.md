Court-Counter App
===================================

An android app that helps you to keep an eye on a basketball match.
I built this as a part of "Udacity Android for Beginners course".

--------------

![whatsapp image 2018-01-08 at 8 04 26 pm](https://user-images.githubusercontent.com/28810075/34675307-3d2a5dee-f4af-11e7-957f-f333c2d9e89c.jpeg)


Pre-requisites
--------------

- Android SDK v22
- Android Build Tools v22.0.1
- Android Support Repository v22.1.1

Getting Started
---------------

This sample uses the Gradle build system. To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
